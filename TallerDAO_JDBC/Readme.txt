Autor: Alba Consuelo Nieto
===========================
El siguiente ejemplo ilustra la implementación de una aplicación en tres capas (presentación, negocio, datos) utilizando el patrón DAO, JavaSwing, JDBC y PostgreSQL. 

El ejemplo:
a) Crea un Empleado con su cargo asociado.
b) Modifica el salario de un empleado.

Para ejecutar el ejemplo se debe:
1. Crear la base de datos rh en PostgreSQL.
2. Cargar el script de la base de datos que está dentro de la carpeta este mismo taller
3. Abrir y compilar el proyecto en Netbeans
4. Ejecutar el main de la aplicación.

Nota:
- Utilizar la versión de Netbeans 8.1 o superior
